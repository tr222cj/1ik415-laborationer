Lab 4 - Mobila enheter

Publicerat av: Tomas, 2014-10-05

Uppgift: Att skapa css för olika skärmstorlekar "responsive" samt mobila enheter.

Uppgiften var relativt enkel jämfört med tidigare uppgifter, antagligen för att allt grundläggande arbete redan var gjort. Det har dock varit ganska mycket "pill" för att få sidorna att se ut som man vill ha dem och till slut lyckades jag :)

Jag återanvänder hela min ursprungliga css och skapar sedan få undantag för att få sidorna att visas snyggt i de situationer jag kunnat förutse och testa i. Den responsiva delen justeras i 4 steg för enheter med skärmar större än 480px:

  1. Skärmstorlekar större än 1000px
  2. Skärmstorlekar mindre än 1000px
  3. Skärmstorlekar mindre än 580px
  4. Skärmstorlekar mindre än 360px

För enheter med skärmar mindre än 481px väljs en femte layout.
